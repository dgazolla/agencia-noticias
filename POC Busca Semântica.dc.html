2. Stack recomendado para a POC

🔢
Embeddings
Converte texto em vetores numéricos
OpenAI text-embedding-3-small ✓ POC
O mais barato do mercado. 1.536 dimensões. Custo de ~R$ 0,10 para embeddar 10.000 artigos.
Alternativa: Cohere embed-multilingual-v3 (bom para pt-BR)
🗃️
Banco Vetorial
Armazena e busca por similaridade
Supabase + pgvector ✓ POC
Free tier generoso (500 MB). PostgreSQL normal + extensão vetorial. Zero configuração de servidor.
Alternativas: Pinecone free tier · Weaviate Cloud · Qdrant Cloud
🤖
LLM (geração da resposta)
Sintetiza a resposta com base nas fontes
Claude 3 Haiku ✓ POC
Mais rápido e barato da família Claude. ~R$ 0,01 por pergunta. Ótimo para RAG.
Upgrade natural: Claude Sonnet · GPT-4o · Gemini Flash
⚙️
Backend da API
Orquestra embeddings + busca + LLM
Python FastAPI · Vercel ✓ POC
10–30 linhas de código. Deploy gratuito no Vercel ou Railway. Alternativa: Node.js + Express.
Sem servidor: Supabase Edge Functions (Deno) também resolve

2
Criar o banco no Supabase e ativar pgvector
Crie um projeto gratuito em supabase.com. No SQL Editor, rode:
-- 1. Ativa a extensão vetorial
create extension vector;

-- 2. Cria a tabela de artigos com coluna de embedding
create table artigos (
  id        text primary key,
  titulo    text,
  editoria  text,
  data      date,
  resumo    text,
  url       text,
  embedding vector(1536)   -- dimensões do text-embedding-3-small
);

-- 3. Função de busca por similaridade (cosine distance)
create or replace function buscar_artigos(
  query_embedding vector(1536),
  top_k int default 5
)
returns table(id text, titulo text, resumo text,
              editoria text, url text, similaridade float)
language sql as $$
  select id, titulo, resumo, editoria, url,
         1 - (embedding <=> query_embedding) as similaridade
  from artigos
  order by embedding <=> query_embedding
  limit top_k;
$$;
3
Script de indexação — embeddar o acervo
Rode uma vez (Python). Pega cada artigo, gera o embedding e salva no Supabase.
# indexar.py — rode apenas uma vez
import json
from openai import OpenAI
from supabase import create_client

client  = OpenAI(api_key="sk-...")
supabase = create_client("https://xxx.supabase.co", "anon-key")

artigos = json.load(open("artigos.json"))

for art in artigos:
    # Texto a embeddar: título + resumo (suficiente para POC)
    texto = f"{art['titulo']}. {art['resumo']}"

    resp = client.embeddings.create(
        model="text-embedding-3-small",
        input=texto
    )
    embedding = resp.data[0].embedding

    supabase.table("artigos").upsert({
        "id":        art["id"],
        "titulo":    art["titulo"],
        "editoria":  art["editoria"],
        "resumo":    art["resumo"],
        "url":       art["url"],
        "embedding": embedding,
    }).execute()

    print(f"✓ {art['titulo'][:50]}")

print("Indexação concluída!")
4
API de consulta — o endpoint do chat
Um único endpoint POST /buscar orquestra a busca semântica e a geração da resposta.
# api.py — FastAPI
from fastapi import FastAPI
from openai import OpenAI
from anthropic import Anthropic
from supabase import create_client

app      = FastAPI()
oai      = OpenAI(api_key="sk-...")
claude   = Anthropic(api_key="sk-ant-...")
supabase = create_client("https://xxx.supabase.co", "anon-key")

@app.post("/buscar")
async def buscar(pergunta: str):

    # 1. Embedda a pergunta do usuário
    emb = oai.embeddings.create(
        model="text-embedding-3-small", input=pergunta
    ).data[0].embedding

    # 2. Busca os 5 artigos mais similares
    resultado = supabase.rpc("buscar_artigos", {
        "query_embedding": emb, "top_k": 5
    }).execute()
    artigos = resultado.data

    # 3. Monta o contexto para o LLM
    contexto = "\n\n".join([
        f"[{i+1}] {a['titulo']}\n{a['resumo']}"
        for i, a in enumerate(artigos)
    ])

    # 4. Gera a resposta com Claude
    resposta = claude.messages.create(
        model="claude-3-haiku-20240307",
        max_tokens=512,
        messages=[{"role": "user", "content": f"""
Você é o assistente do portal de notícias da Agência CNI.
Responda à pergunta em até 3 frases, citando as fontes [1], [2]...

Artigos encontrados:
{contexto}

Pergunta: {pergunta}"""}]
    )

    return {
        "resposta": resposta.content[0].text,
        "fontes":   artigos
    }
5
Conectar ao chat do portal
Substituir o matching manual do protótipo por uma chamada à API. Duas linhas de JavaScript.
// No chat do portal — substitui a função doAsk()
async function doAsk(pergunta) {
  const res = await fetch("https://sua-api.vercel.app/buscar", {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({ pergunta })
  });
  const { resposta, fontes } = await res.json();

  // Exibe resposta + lista de artigos fonte
  setAnsText(resposta);
  setFontes(fontes);  // renderiza as matérias retornadas
}
